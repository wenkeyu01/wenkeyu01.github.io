This repository hosts the source code for the homepage of **Wen Keyu**, a Ph.D. student specializing in Operations Research at the AMSS, CAS.

The website is designed to be a clean, professional, and responsive portfolio to showcase academic background, research interests, publications, and personal photography.

## âœ¨ Features

- **Professional Academic Design**: A minimalist layout featuring a classic deep blue and white color scheme, tailored for academic profiles.
- **Fully Responsive**:
  - **Desktop**: Two-column layout with a fixed sidebar and wide content area; gallery displays in 6 columns.
  - **Mobile**: Optimized single-column layout with adjusted avatar size, touch-friendly navigation, and a 2-column gallery for better visibility on small screens.
- **Lightweight & Fast**: Built with pure **HTML5** and **CSS3**. No external JavaScript frameworks or heavy libraries are used, ensuring lightning-fast load times.
- **Bilingual Support**: Key information, navigation menus, and buttons include both English and Chinese translations.
- **Interactive Details**: Includes smooth fade-in animations, hover effects for links and images, and a custom-designed "split-capsule" CV download button.
